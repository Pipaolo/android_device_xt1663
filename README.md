Motorola M (XT1633)
==============

This branch is for building of the LineageOS 14.1.

# About Device

[Moto M]

# Build Information

### Working:
 * Rotation
 * Micro SD support
 * Wi-Fi
 * Audio ( Ultra Low Volume )
 * Offline Charging
 * Auto brightness
 * Offline charging

### Not Working:
 
 * Wi-Fi tethering
 * GPS
 * Bluetooth
 * AGPS
 * Vibration
 * All sensors
 * Flashlight (statusbar and camera)
 * Camera rear/front (video and photo) 
 * RIL for SMS/calls/internet/LTE 
 * MTP 
 * LED Notifications
 * FM Radio
 * Camera HDR
 * Youtube
 * HW Decoder
 * Hotspot (Open Hotspot)
